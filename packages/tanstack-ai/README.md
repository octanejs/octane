# @octanejs/tanstack-ai

Octane bindings for [TanStack AI](https://tanstack.com/ai). Ports the
`@tanstack/ai-react` hook layer to Octane while reusing the framework-agnostic
`@tanstack/ai` and `@tanstack/ai-client` packages unchanged.

Status: alpha. See `status.json` for the supported surface and divergences.
