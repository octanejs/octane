# What is vyre?

[![status: alpha](https://img.shields.io/badge/status-alpha-orange)](https://www.npmjs.com/package/vyre)
[![npm version](https://img.shields.io/npm/v/vyre?logo=npm)](https://www.npmjs.com/package/vyre)
[![npm downloads](https://img.shields.io/npm/dm/vyre?logo=npm&label=downloads)](https://www.npmjs.com/package/vyre)

Vyre is a fast, TypeScript-first UI framework, and the successor to
[Inferno](https://github.com/infernojs/inferno). It gives you the React API you
already know, a compiler that keeps the runtime small and fast, and no rules of
hooks, so you can call hooks conditionally. This package ships both the runtime
and the compiler, with the compiler exposed at `vyre/compiler`.

For the full story, see the
[main README](https://github.com/vyre-ts/vyre#readme).

Vyre is alpha software. It is ready to try, but not yet ready for production.
